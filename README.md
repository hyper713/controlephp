# controlephp

## Project Setup & Run (if you use docker)
1. clone/download the project.
2. Run `docker-compose up`
3. Import the database `dumps/VEHICULE_CIRCULATION.sql`
4. go to [localhost](localhost:80)
5. enjoy :)

## Project Setup & Run (if you don't use docker)
1. clone/download the project.
2. copy `dumps` and `src` only
3. Import the database `dumps/VEHICULE_CIRCULATION.sql`
4. edit database file `src/database.php`
5. test the app on your machine using `src/index.php`
6. enjoy :)
